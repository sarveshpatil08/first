Pandas (Data Analysis Library)

Import:
import pandas as pd

Creating DataFrame
data = {
    "City":["Mumbai","Delhi","Pune"],
    "Sales":[200,300,250]
}
df = pd.DataFrame(data)

Viewing Data - First rows
df.head()

Last rows
df.tail()

Information
df.info()

Summary statistics
df.describe()

Select column
df["Sales"]

Multiple columns
df[["City","Sales"]]

Filtering Data
Example: sales greater than 200
df[df["Sales"] > 200]

GroupBy
example dataset :
| City   | Sales |

df.groupby("City")["Sales"].mean()
